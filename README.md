# Men vs Women Image Classification

### AI / Deep Learning Project | Python, CNN, TensorFlow

---

## Overview

This project focuses on building an **image classification model** capable of distinguishing between **male and female facial images** using **Convolutional Neural Networks (CNNs)**.
The system processes facial images and automatically predicts the gender category based on learned visual patterns.

The goal of this project is to demonstrate the application of **deep learning techniques in computer vision**, particularly for image classification tasks involving facial features.

---

## Key Features

* Deep learning model for **gender classification from images**
* Implementation using **Convolutional Neural Networks (CNN)**
* Image preprocessing including **resizing, normalization, and augmentation**
* Improved classification accuracy through **model optimization techniques**
* Ability to handle variations in **facial expressions, clothing, and backgrounds**

---

## Tech Stack

* **Programming Language:** Python
* **Deep Learning Framework:** TensorFlow
* **Model Architecture:** Convolutional Neural Networks (CNN)
* **Libraries Used:** NumPy, OpenCV, Matplotlib

---

## Project Workflow

The project follows a standard **machine learning workflow**:

1. **Dataset Collection**

   * Gathered facial image datasets containing male and female images.

2. **Data Preprocessing**

   * Resized images for consistent input dimensions.
   * Normalized pixel values for improved training performance.
   * Applied augmentation techniques to improve generalization.

3. **Model Development**

   * Designed and trained a **CNN-based classification model**.
   * Extracted facial features using convolution and pooling layers.

4. **Model Evaluation**

   * Tested the trained model on unseen images.
   * Measured performance using classification accuracy.


## Installation

Clone the repository:

```bash
git clone https://github.com/Sathwika-Thokala/MEN-WOMEN-CLASSIFICATION.git
cd MEN-WOMEN-CLASSIFICATION
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## How to Run the Project

Train the model:

```bash
python train_model.py
```

Run prediction on a new image:

```bash
python predict.py
```

The system will analyze the image and classify it as **Male** or **Female**.
